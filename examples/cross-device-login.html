Sign in with Signet — cross-device demo

Demonstrates the relay-delivery flow: this page generates a session keypair, renders a QR encoding the Signet app URL with relay + sessionPubkey, and waits for the signed auth response via NIP-17 unwrap.

Before running:
1. Run a local Nostr relay at ws://localhost:7777 (e.g. nostr-rs-relay or similar).
2. Serve the Signet app dev server at http://localhost:5174.
3. Serve this page via any static server (e.g. python3 -m http.server 8080) and open on your desktop browser.
4. On your phone, open the QR link manually (or scan if you render a real QR) — it points to your desktop http://localhost:5174 dev server (use your network IP in place of localhost for a real phone test).
